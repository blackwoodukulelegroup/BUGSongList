Adding songs to the BUG Song List

There are two ways to add songs to the BUG song list; as a link (e.g. to a Scorpex PDF), or as a custom PDF

Adding links to external sites/PDFs

  1. Open the BUG chord Charts spreadsheet
    NB - You need to log on as a Google user with read/write access
  2. Go to the Songs tab
  3. Add an entry as follows:
    • Title - Title of the song e.g. 26 Miles
    • Artist - Who wrote, or most notably performed the song e.g. The Four Preps
    • AttributeName - Chart (key) e.g. Chart (C) if the song is in C major
    • AttributeType - URL
    • AttributeValue - The link to the PDF or song page e.g. https://scorpexuke.com/pdffiles/26_Miles.pdf
    • EntryType - ManualEntry
  4. The sheet should save automatically (no need to do File/Save)
  5. Open up the Song List and check that the new entry appears.
    If you can't see the entry, try doing a full refresh by pressing ctrl + F5.
    If you still can't see the entry, re-open the spreadsheet and check the entry is correct.

Adding custom PDFs

  1. PDFs need to be named in this format: [Song Title] - [Artist Name] - BUG ([key]).PDF
    e.g. 26 Miles - The Four Preps - BUG (C).pdf
    NB. PDFs can be renamed before, or after uploading into Google Drive.
  2. Upload the PDF file to BUG Chord Charts folder in the Blackwood Uke Group Google Drive.
    NB - You need to log on as a Google user with read/write access
  3. Open the Google Drive Reindex script and click the Click to start indexingstrong> button.
  4. Open up the Song List and check that the new entry appears.
    If you can't see the entry, try doing a full refresh by pressing ctrl + F5.
    If you still can't see the entry, or it doesn't look right, check the name is in the right format.
